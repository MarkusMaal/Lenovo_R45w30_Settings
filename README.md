# Lenovo Legion R45w-30 TUI settings

This project gives you the ability to change the settings for this monitor without the need for Artery software. It even support some of the manufacturer specific commands.

Requirements: ddcutil (allows the program to communicate with the monitor)

The following non-standard settings are available:

- Game Mode
- Over drive
- True split
- Input source: USB-C
- Administrative lock
- Overlays (FPS counter, Countdown, Crosshair, System stats, Stopwatch)
- DCR
- PbP (enable/disable and input sources)
- HDR
